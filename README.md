# BIMI Website

This is a Next.js project bootstrapped with `pnpm`.

## Getting Started

First, ensure you have `pnpm` installed globally:

```bash
npm install -g pnpm
```

Then, install the project dependencies:

```bash
pnpm install
```

Finally, run the development server:

```bash
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Environment Variables

This project uses Supabase for its backend. You will need to set up the following environment variables. Create a `.env.local` file in the root of your project and add the following, filling in the appropriate values from your Supabase project:

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

You can find an example in `.env.example`.
